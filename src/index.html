Front-end

O front-end refere-se à parte visível e interativa de um site ou aplicação com a qual os usuários interagem diretamente. É a camada da interface do usuário (UI) que se comunica com o back-end (a parte do sistema que processa dados e lógica). O trabalho do desenvolvedor front-end é garantir que a interface seja intuitiva, responsiva (adaptando-se a diferentes dispositivos) e funcional, proporcionando uma boa experiência ao usuário.

UX (User Experience) - UI (User Interface)

UX refere-se à forma como um usuário interage com um produto, serviço ou sistema e as emoções e percepções geradas por essa interação. UI foca na aparência visual e no design dos elementos com os quais o usuário interage.

Data Science

Data Science é um campo interdisciplinar que combina técnicas de estatística, computação e conhecimento específico de domínio para extrair conhecimento e insights úteis a partir de dados estruturados e não estruturados.

Back-end

O back-end refere-se à parte "invisível" de um sistema ou aplicação responsável pelo processamento, armazenamento e gestão dos dados. Ele funciona nos bastidores, garantindo que a lógica de negócio, banco de dados e servidores trabalhem em conjunto para suportar a interação que ocorre no front-end (a interface visível ao usuário).
